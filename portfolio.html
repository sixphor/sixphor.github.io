I developed a travel list app using React, designed to streamline the packing process for users planning their trips. The app allows users to easily add items to their packing list and specify the quantity of each item, making it ideal for travelers who want to stay organized and avoid forgetting essential items.
In addition to adding items, the app provides a user-friendly interface where items can be removed or updated as needed. Each item on the list features a checkbox, allowing users to mark items as "packed" once they've been taken care of. This provides a clear visual indicator of their packing progress.
A key feature of the app is its real-time tracking of packing completion. As users check off items, the app dynamically calculates and displays the percentage of packed items compared to the total, giving users a clear sense of how much progress they've made in preparing for their journey. This functionality offers a comprehensive overview at a glance, ensuring travelers feel prepared without needing to scroll through the entire list.

This project is a full-stack workout tracking web application, developed using the MERN stack (MongoDB, Express.js, React, Node.js). The app allows users to create personalized workout routines, track progress, and log their exercises with details such as sets, reps, and weight.

On the frontend, I implemented React for building a responsive and dynamic user interface, providing users with smooth navigation and interactive features like workout timers and exercise history tracking. The design focuses on simplicity and functionality, allowing users to easily create, edit, and view their workout plans.

On the backend, I used Node.js and Express to handle user authentication, session management, and data storage. MongoDB was utilized as the database to store user profiles, workout logs, and performance metrics. The backend supports secure operations, with JWT-based authentication to protect user data.
