AR/VR Applications

VR Escape Room: "Your Heart, Your Reality"

Using Unity, I created my escape room game "Your Heart, Your Reality". In this game, the player must solve puzzles in order to escape the rooms and complete the game. The game was created using the Unity engine, creating elements and objects and placing them in the scene to form an environment. In this project, C# was used to create the game's features such as opening the doors, picking up objects, clicking buttons.

Using a combination of Node.js, PHP and MySQL, I created a storybook app, "No Book, No Life". This application allows people to create a user with a name, profile picture and description. It also allows the main part, creating books, with a title, author, book cover, and synopsis. The book list and user list will be updated accordingly when one is created or deleted. Throughout the creation of the app, I discovered how to implement multiple features and how to link to a local database, where all the book and user information are stored.

React App: Pizza Shop

Using React, I created a pizza ordering app. This app created using the JSX. This app displays a pizza menu with various detais such as price amnd ingredients. There is also a feature based on the real-time. When it is in a certain time range, the app will be in a different display to show open or closed.

Design

Avatar creations

Using Adobe Illustrator, I created some avatar designs. These designs were created from scratch and they are variants of the same avatar, with different hair, clothes, expressions and more. I used a variety of tools to create this design and learnt a lot of use techniques along the way. Each body part was made on its own before then combining them to form a human character.
